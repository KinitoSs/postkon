# Postkon

## Запуск проекта

Чтобы запустить сайт, нужно клонровать репозиторий, открыть командную строку в папке проекта, написать `python manage.py runserver` или `py manage.py runserver`, `python3 manage.py runserver`.

В адресной строке в браузере написать, к примеру, `localhost:8000/users/kin`.

## Навигация по проекту

* HTML-файлы находятся по пути: `postkon/postkon_webapp/templates/postkon_webapp/`.
* CSS-файлы находятся по пути: `postkon/postkon_webapp/static/postkon_webapp/css/`.
* JS-файлы находятся по пути: `postkon/postkon_webapp/static/postkon_webapp/js/`.
* Картинки находятся по пути: `postkon/postkon_webapp/static/postkon_webapp/img/`.
* Urls находятся по пути: `postkon/postkon_webapp/urls.py`.
* Views находятся по пути: `postkon/postkon_webapp/views.py`.
* Модели находятся по пути: `postkon/postkon_webapp/models.py`.
